## 项目简述
本项目是一个基于bootstrap框架搭建的一个动态表格。</br>
本项目采用了MVC开发模式，前端采用Bootstrap+jQuery开发技术,后端采用了Express框架+MongoDB开发技术,其中MongoDB使用了Mongoose进行驱动。</br>
</br>
## 启动方法
``` bash
# 安装后端所有依赖
npm install

# 启动后端服务,服务启动成功后，直接访问http://localhost:3000即可。
npm start

```
